================================================================================
NGSGEOID - NOAA NGS Geoid Height Grids
================================================================================

.. shortname:: NGSGEOID

.. built_in_by_default::

GDAL supports reading NOAA NGS geoid height grids in binary format (.bin
files). Those files can be used for vertical datum transformations.

Driver capabilities
-------------------

.. supports_georeferencing::

.. supports_virtualio::
