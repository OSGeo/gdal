RIEGL RDB 2

This driver reads from *.mpx files in the RDB format, the in-house format used by RIEGL Laser Measurement Systems GmbH

The driver relies on the RDB library, which can be downloaded here. The minimum version required is 2.2.0.

Use --with-rdb when running configure in order to enable the rdb driver for GDAL.

Currently *.mpx files are using the Web Mercator projecton, EPSG:3857.
