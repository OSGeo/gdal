PLScenes (Planet Labs Scenes/Catalog API)

(GDAL/OGR >= 2.0 for V0 API, GDAL/OGR >= 2.1 for V1 API, GDAL/OGR >= 2.2 for Data V1 API)

This driver can connect to Planet Labs Scenes v0/Catalog v1/Data V1 API. GDAL/OGR must be built with Curl support in order for the PLScenes driver to be compiled.

Please consult the dedicated pages for each version of the API:

  • V0 API (deprecated)
  • V1 API (deprecated)
  • Data V1 API (current)
