GDAL is registered as reference implementation for OGC KML 2.2, OGC GMLJP2 v2 and OGC GeoTIFF 1.1

We have scripts to generate the files to submit to https://cite.opengeospatial.org/teamengine

- kml_generate_test_files.py: generates test_ogrlibkml.kml, test_ogrlibkml.kmz, test_ogrlibkml_update.kml and test_superoverlay.kmz
- generate_ogc_geotiff_1_1_samples.sh: generates epsg26711_geotiff1_1.tif and epsg4326_3855_geotiff1_1.tif
- generate_ogc_gmljp2_v2_samples.sh: generates byte_gmljp2_v2.jp2
