================================================================================
"gdal raster overview" sub-command
================================================================================

.. versionadded:: 3.11

.. only:: html

    Manage overviews of a raster dataset

.. Index:: gdal raster overview

Synopsis
--------

.. code-block::

    Usage: gdal raster <SUBCOMMAND>
    where <SUBCOMMAND> is one of:
      - add:    Adding overviews.
      - delete: Deleting overviews.
