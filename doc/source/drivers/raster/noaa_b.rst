================================================================================
NOAA_B -- NOAA GEOCON/NADCON5 .b format
================================================================================

.. versionadded:: 3.7

.. shortname:: NOAA_B

.. built_in_by_default::

GDAL supports reading geodetic grids in the NOAA .b binary format used for
GEOCON and NADCON5 grids.

Driver capabilities
-------------------

.. supports_georeferencing::

.. supports_virtualio::

--------
